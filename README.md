# 🎂 Zander's Birthday Gift Website

A personalised, interactive birthday gift webpage built with pure HTML, CSS, and JavaScript — no backend, no installs, works in any browser on any device.

---

## 🎯 What This Page Does

When Zander opens the link, he sees:

### 1. Splash Screen
- Full-screen birthday intro with animated glowing title and twinkling stars
- A **"Open Your Gift"** button that triggers a fireworks explosion and reveals all sections

### 2. 🏀 Basketball Star Cards
- Three flip cards: **Stephen Curry**, **LeBron James**, and a custom **"Zander — Ultra Rare"** card
- Hover (desktop) or tap (mobile) each card to flip it and reveal a personalised message on the back

### 3. 🔫 CSGO Birthday Case
- A clickable unbox animation with suspense
- Randomly reveals one of four birthday-exclusive weapon skins with rarity labels and fireworks

### 4. ⛳ Golf Course Blessings
- Green-themed section with a personalised golf birthday message
- **"Take Your Shot!"** button triggers a Hole-In-One celebration with fireworks

### 5. 🎂 Final Message
- A warm, personal closing birthday message for Zander
- Scroll-triggered fade-in animation

---

## 🛠 Tech Stack
- **HTML5** — structure
- **CSS3** — animations, flip cards, fireworks particles, responsive layout
- **Vanilla JavaScript** — interactivity, scroll observer, fireworks engine

## 🚀 Deployment
Hosted on **GitHub Pages** — accessible via any browser, zero installation required.
